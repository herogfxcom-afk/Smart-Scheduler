# Инструкция по откату (Rollback Guide)

Если что-то пошло не так, вы можете вернуться к этой стабильной версии, используя тег:
**`v1.0.3-stable-aligned-deletion`**

## Что входит в эту версию:
1.  **Исправленное удаление встреч**: Теперь ID событий Google/Outlook сохраняются для всех участников, позволяя гостям очищать календари.
2.  **Ровная сетка календаря**: Групповой календарь выровнен по 30-минутным интервалам (Snap to Grid).
3.  **Стабильный деплой на Vercel**: Полностью рабочая синхронизация и API.
4.  **Умная отмена встреч**: Логика сохраняет слоты участников при отмене создателем.
5.  **Inline Magic Sync**: Красивая плашка при поиске бота в чатах.

### Другие способы восстановления:

1. **GitHub**: Перейдите в раздел "Tags" на вашем GitHub и скачайте ZIP-архив версии `v1.0.3-stable-aligned-deletion`.
2. **Vercel**: В панели управления Vercel вы всегда можете найти деплоймент от этого момента и нажать **"Redeploy"** или **"Promote to Production"**.
